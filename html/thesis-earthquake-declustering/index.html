Problem Statement

What is Earthquake Declustering?

Earthquake catalogues contain two types of events: independent background earthquakes that occur randomly due to tectonic stress, and dependent events such as aftershocks that are triggered by previous earthquakes. Declustering is the process of separating these two types of events.

This separation is critical for seismic hazard assessment, earthquake forecasting, and understanding long-term tectonic behavior. Background seismicity follows a Poisson process (random, independent events), while clustered events show spatial and temporal dependencies.

Limitations of Traditional Methods

Classical declustering techniques face several challenges:

  • Fixed Parameters: Window-based methods like Gardner-Knopoff (1974) and Reasenberg (1985) use predetermined time and distance windows that may not work equally well across different seismic regions.
  • Overlapping Clusters: In complex tectonic environments like New Zealand, aftershock sequences often overlap with independent events, making separation difficult.
  • Subjective Tuning: Results depend heavily on chosen thresholds and parameter values, reducing reproducibility and objectivity.
  • Statistical Assumptions: Model-based approaches like ETAS (Epidemic-Type Aftershock Sequence) assume constant background rates and can be biased by incomplete catalogues.

Why Machine Learning?

Machine learning models can automatically learn complex, nonlinear patterns from data without requiring fixed rules or thresholds. By training on synthetic earthquake catalogues where true event types are known, ML algorithms can:

  • Adapt to different seismic environments
  • Capture multidimensional relationships between time, space, and magnitude
  • Provide objective, reproducible classifications
  • Generalize across regions without manual parameter tuning

Why New Zealand?

New Zealand sits at the boundary between the Pacific and Australian tectonic plates, experiencing complex seismicity through:

  • Hikurangi Subduction Zone: Pacific Plate subducting beneath North Island
  • Alpine Fault: Major strike-slip fault system in South Island
  • Crustal Faulting: Distributed deformation across both islands

This tectonic complexity results in frequent, overlapping aftershock sequences‚Äîmaking it an ideal testbed for advanced declustering methods.

Nearest-Neighbour Distance (NND) Method

The NND approach transforms earthquake occurrences into a rescaled space-time-magnitude framework. For each event pair (i, j), we compute:

Rescaled Proximity Distance: Œ∑ij = Tij √ó Rij

where:

  • Tij = rescaled time interval
  • Rij = rescaled spatial distance

The distribution of log(Œ∑) typically shows two distinct peaks: one for tightly clustered aftershocks (low Œ∑) and another for independent background events (high Œ∑).

ETAS Model for Synthetic Data Generation

The Epidemic-Type Aftershock Sequence (ETAS) model treats earthquakes as a branching process where each event can trigger its own aftershocks. The model provides labeled training data by explicitly marking which events are background and which are triggered.

Key ETAS parameters:

  • Œº: Background seismicity rate
  • K, Œ±: Productivity parameters (aftershock generation)
  • c, p: Omori-Utsu temporal decay parameters
  • D, Œ≥, q: Spatial decay parameters

Machine Learning Models

Four supervised classifiers were trained and compared:

Random Forest (RF)

Ensemble of decision trees using bootstrap aggregating. Robust to overfitting and provides feature importance rankings.

Support Vector Machine (SVM)

Finds optimal hyperplane separating background and triggered events in high-dimensional feature space.

Gradient Boosting (GB)

Sequential ensemble that iteratively corrects misclassifications, achieving high recall for triggered events.

XGBoost

Optimized gradient boosting with regularization, achieving the best overall performance (97.44% accuracy).

Model Training Protocol

  • Data Split: 80% training, 20% testing from synthetic ETAS catalogue
  • Cross-Validation: 5-fold CV to prevent overfitting
  • Hyperparameter Tuning: Grid search optimization for each model
  • Evaluation Metrics: Accuracy, Precision, Recall, F1-Score
  • Feature Standardization: Z-score normalization to remove scale effects

Feature Importance Analysis

Analysis across all models revealed consistent feature importance rankings:

  1. N‚Å∫ (Siblings Count): Most influential predictor‚Äîevents with many siblings are likely part of aftershock sequences
  2. R‚Å∫ (Rescaled Distance): Spatial proximity to nearest neighbor strongly indicates clustering
  3. T‚Å∫ (Rescaled Time): Temporal closeness helps identify recently triggered events
  4. dm‚Å∫ (Magnitude Difference): Larger parent magnitudes increase triggering probability
  5. nchild (Offspring Count): Events that trigger others are more likely mainshocks
  6. nparent: Multiple potential parents indicate complex overlapping sequences

This ranking confirms physical expectations: aftershocks cluster tightly in space and time around larger parent events, validating the NND framework's effectiveness.

Concluding Remarks

This research demonstrates that machine learning offers a powerful, objective alternative to traditional earthquake declustering methods. By combining the physical insights of the ETAS model with the pattern-recognition capabilities of modern ML algorithms, we achieved classification accuracy exceeding 97% on synthetic data and produced physically consistent results when applied to New Zealand's complex seismic catalogue.

The declustered catalogue provides a clearer picture of New Zealand's background seismicity and aftershock patterns, directly supporting improved seismic hazard assessment in this tectonically active region. The framework's adaptability and objectivity make it a valuable tool for seismologists worldwide working to understand earthquake clustering and improve public safety.
